## stack sync

Refresh cached PR metadata and inspect or apply safe repairs

### Synopsis

Refresh cached PR metadata from GitHub and either report or apply clean, classified repairs. Ambiguous cases stop for review.

```
stack sync [flags]
```

### Examples

```
stack sync
stack sync --apply
```

### Options

```
      --apply   Apply clean merged-parent repairs
  -h, --help    help for sync
```
